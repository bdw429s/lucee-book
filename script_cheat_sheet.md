# Script Quick Reference

## If
```
if(condition){
  doSomething();
}
```

## If/else
```
if(condition){
  doSomething();
} else {
  doSomethingElse();
}
```

## If/else if
```
if(condition){
  doSomething();
} else if(condition2) {
  doSomethingElse();
} else {
  doSomethingMore();
}
```
##For Loop
```
for(var i = 1; i <= 5; i=i+1){
  echo('my text');
}
```

##For IN Loop Array
```
for(var i IN myArray){
  //Outputs the value of the array element
  echo(i);
}
```
